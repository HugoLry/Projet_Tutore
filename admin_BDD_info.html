Administrateur(trice) de base de données

Avez-vous déjà envisagé de devenir administrateur ou administratrice de base de données ? Un métier méconnu du grand public, l'administrateur/administratrice de base de données joue un rôle prépondérant dans la structure informatique d'une entreprise. L’administrateur/administratrice de base de données est le gestionnaire des milliers d'informations numériques : adresses clients, tarifs, statistiques, stocks de marchandises, documents.Il ou elle met ces données (Datas) à la disposition de l’entreprise qui l'emploie et en assure la disponibilité, la cohérence, l’accessibilité, la qualité et la sécurité.

Diplôme et formations

Ce métier requiert une formation initiale en développement avec une spécialisation en bases de données ou d'une formation généraliste orientée systèmes et réseaux. Le métier d'administrateur/administratrice de base de données est accessible à partir du niveau bac +2 avec quelques années d'expérience, mais les employeurs préfère les candidats avec un niveau bac +5 ou un diplôme issue d’une école d’ingénieurs. À partir d'un niveau Bac+2, il est possible de se spécialiser dans une mission et de compléter sa formation dans l'entreprise. De 2 à 5 années d'études supérieures et une expérience dans le développement informatique est souhaitable. Information notable : une certification de qualification professionnelle délivrée par les éditeurs peut être requise par certaines entreprises.

Perspective d'évolution

Le rôle de l'administrateur ou administratrice de base de données évolue progressivement vers le développement des réseaux informatiques. Avec de l’expérience, tu pourra évoluer vers plusieurs postes tels que : Responsable informatique, Responsable stockage, Responsable infrastructure ou Responsable d’équipe. Si tu le souhaites, tu pourras te tourner vers les métiers d'architecte système, de Consultant/Consultante en optimisation de bases de données ou encore de Chef/Cheffe de projet.
